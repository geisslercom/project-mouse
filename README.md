# ProjectMouse

## Installing

	git clone "https://git.webvariants.de/benjamin-geissler/projectMouse.git"

	cd projectMouse

	npm install -g


## Usage

### Adding a Path

	pm --save [PATH]

### Removing a Path

	pm --remove [PATH]

### Getting all Projects

	pm

### Getting Projects Bower Stack

	pm -b

### Getting Projects NPM Stack

	pm -n
